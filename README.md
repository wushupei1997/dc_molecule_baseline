## 1.数据说明  
**train.csv**

|       字段名       |  类型  |                        说明                         |
| :----------------: | :----: | :-------------------------------------------------: |
|         ID         |  整型  |                      样本编号                       |
| Molecule_max_phase |  整型  |                   分子的最长位相                    |
|  Molecular weight  | 浮点型 |                       分子量                        |
|   RO5_violations   |  整型  |             违反新药5规则（RO5）的数量              |
